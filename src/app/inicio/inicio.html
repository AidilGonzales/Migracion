¡Bienvenido a mi proyecto final! 🎓

Soy Aidil Gonzales, estudiante de Ingeniería de Sistemas en la Universidad. Este proyecto fue desarrollado como parte del curso de Programación Web, donde aplico mis conocimientos en Angular, HTML, CSS y TypeScript.

La finalidad de esta aplicación es mostrar de manera práctica cómo crear un sitio moderno, con componentes reutilizables, enrutamiento y comunicación entre módulos. Es un proyecto pensado para aprender y demostrar mis habilidades técnicas y creativas.
